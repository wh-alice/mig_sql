# Browse Data Source Dialog Box (Analysis Services - Multidimensional Data)
  Use the **Browse Data Source** dialog box in [!INCLUDE[ssBIDevStudioFull](../a9notintoc/includes/ssbidevstudiofull-md.md)] to view the first 1,000 records of the selected table.  
  
## Options  
 **Data Source Table**  
 Displays the first 1,000 records of the selected table. All data is read-only.
