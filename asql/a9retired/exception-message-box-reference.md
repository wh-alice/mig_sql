# Exception Message Box Reference
  The exception message box is implemented by the <xref:Microsoft.SqlServer.MessageBox.ExceptionMessageBox> class in the Microsoft.ExceptionMessageBox.dll assembly. The default installation folder for this assembly is C:\Program Files (x86)\Microsoft SQL Server\130\SDK\Assemblies. The exception message box is documented in the <xref:Microsoft.SqlServer.MessageBox> namespace reference documentation. The following samples use the <xref:Microsoft.SqlServer.MessageBox.ExceptionMessageBox> object during exception handling.
