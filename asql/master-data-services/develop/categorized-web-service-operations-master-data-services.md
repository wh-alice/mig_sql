# Categorized Web Service Operations (Master Data Services)
  The [!INCLUDE[ssMDSshort](../../analysis-services/includes/ssmdsshort-md.md)] web service contains a complete set of operations that let you write code to control all of the features that [!INCLUDE[ssMDSmdm](../../database-engine/install/windows/includes/ssmdsmdm-md.md)] does through its user interface. The web service operations are defined by the <xref:Microsoft.MasterDataServices.IService> interface and are implemented as methods in the <xref:Microsoft.MasterDataServices.ServiceClient> class. This topic groups the web service operations into conceptual categories to help you understand how to use the web service API.  
  
## Model Operations  
 These operations are used to create, update, and delete models, as well as to operate on all the contents of a model, such as entities, hierarchies, and versions. For more information, see [Models &#40;Master Data Services&#41;](../../master-data-services/models-master-data-services.md).
