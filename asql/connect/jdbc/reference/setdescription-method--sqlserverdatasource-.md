[!INCLUDE[Driver_JDBC_Download](../../../connect/jdbc/includes)]

## Syntax  
  
```  
  
public void setDescription(java.lang.String description)  
```  
  
#### Parameters  
 *description*  
  
 A **String** that contains the description.
