# Connect to a Registered Server (SQL Server Management Studio)
  This topic describes how to connect to a registered server in [!INCLUDE[ssCurrent](../../a9notintoc/includes/sscurrent-md.md)] by using [!INCLUDE[ssManStudioFull](../../a9notintoc/includes/ssmanstudiofull-md.md)]. If the server is not running or cannot be found on the network, Registered Servers displays an error.  
  
##  <a name="SSMSProcedure"></a>  
  
#### To connect to a registered server  
  
1.  In Registered Servers, right-click a server, point to **Connect**, and then click **Object Explorer**.
