# Select a Data Source (Linked Object Wizard)
  Use the **Select a Data Source** page to select the [!INCLUDE[msCoName](../a9notintoc/includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../a9notintoc/includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../a9notintoc/includes/ssasnoversion-md.md)] data source that contains the objects from which to link or import.  
  
## Options  
 **Analysis Services data sources**  
 Select the data source that contains the object from which to link.  
  
 **Data source properties**  
 Displays the property names and values of the relevant properties on the data source.  
  
 **New Data Source**  
 Create a new data source with the Data Source Wizard.
