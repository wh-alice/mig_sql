# Go To Dialog Box (SSAS)
  The **Go To** dialog box enables you to jump to a specific column in a table.  
  
## UIElement List  
  
|||  
|-|-|  
|Term|Definition|  
|**Go to**|Select a column from the **Go to** list, and then click **OK**.|
