# MSSQL_REPL-2147200928
    
## Message Details  
  
|||  
|-|-|  
|Product Name|SQL Server|  
|Event ID|-2147200928|  
|Event Source|MSSQLServer|  
|Symbolic Name||  
|Message Text|The version of [!INCLUDE[ssNoVersion](../../advanced-analytics/r-services/includes/ssnoversion-md.md)] running at the Subscriber is incompatible with the publication. Either upgrade the Subscriber to '%1' or change the publication compatibility level of the publication to match the version of [!INCLUDE[ssNoVersion](../../advanced-analytics/r-services/includes/ssnoversion-md.md)] currently running at the Subscriber.|  
  
## Explanation  
 The Publisher of the specified publication has a publication compatibility level higher than the current Subscriber.  
  
## User Action  
 Either upgrade the Subscriber or re-create the publication with a compatibility level that matches the current version of the Subscriber.
